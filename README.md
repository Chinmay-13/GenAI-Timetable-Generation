# Timetable Generation System

AI-powered academic timetable generator for CSE Department, 3rd Semester, 12 sections (A–L), 5 courses.

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# Optional AI features (RAG + LangChain agent)
pip install sentence-transformers faiss-cpu langchain langchain-google-genai

# 2. Set up GROQ API key
copy .env.example .env
# Edit .env and set: GROQ_API_KEY=your_key_here

# 3. Generate timetable
python run_all.py

# 4. Launch web dashboard
streamlit run app.py
```

## Full Dependency Install

pip install pandas>=2.0 ortools>=9.0 python-dotenv>=1.0 google-generativeai>=0.8 streamlit>=1.25 altair>=4.2 openpyxl>=3.0 networkx>=3.0

## Streamlit App — 5 Pages

| Page | Description |
|---|---|
| **1. Dashboard** | Summary stats, faculty load chart, Refresh Data button |
| **2. View Timetables** | Browse section + faculty timetable grids |
| **3. Substitute Finder** | Find substitutes for absent faculty + load-swap planning |
| **4. AI Assistant** | Gemini chat with RAG-augmented answers |
| **5. AI Agent** | LangChain tool-use agent — queries real files directly |

## 6-Phase Pipeline

| Phase | File | Description |
|---|---|---|
| 0 | `src/phase0/` | Load CSVs + validate assignments |
| 1 | `src/phase1/` | Build faculty-section assignment map |
| 2 | `src/phase2/` | Lock lab slots (P7-P9) from `lab_allotment.csv` |
| 3 | `src/phase3/` | OR-Tools CP-SAT theory scheduling |
| 4 | `src/phase4/` | Export section/faculty CSVs + summary report |
| 5 | `src/phase5/` | AI explainer, chat, substitute/swap planning, RAG, agent |
